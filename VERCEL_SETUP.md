# Настройка Яндекс Карт API на Vercel

## 1. Получение API ключа Яндекс Карт

1. Перейдите на https://developer.tech.yandex.ru/
2. Войдите или зарегистрируйтесь
3. Создайте новое приложение
4. В настройках приложения включите "JavaScript API и HTTP Геокодер"
5. Скопируйте API ключ

## 2. Настройка на Vercel

### Способ 1: Через Vercel Dashboard (рекомендуется)

1. Откройте ваш проект на https://vercel.com/dashboard
2. Перейдите в Settings → Environment Variables
3. Добавьте новую переменную:
   - **Name**: `REACT_APP_YANDEX_API_KEY`
   - **Value**: ваш-ключ-от-яндекс-карт
4. Нажмите Save
5. Сделайте redeploy проекта

### Способ 2: Через Vercel CLI

```bash
# Установите переменную окружения
vercel env add REACT_APP_YANDEX_API_KEY

# Или импортируйте из .env файла
vercel env pull .env.production
```

## 3. Проверка работы

После деплоя проверьте консоль браузера - должны появиться реальные адреса при вводе.

## 4. Альтернативы (если не хотите настраивать Яндекс)

Приложение работает и без API ключа, но с упрощенными подсказками:
- `ул. [название]`
- `пр. [название]`
- `пл. [название]`
- `кафе [название]`
- `ресторан [название]`
- и т.д.

## 5. Другие варианты API

Если Яндекс не подходит, можно использовать:

- **Mapbox** (есть бесплатный tier) - нужно заменить в коде
